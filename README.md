## Challenges & How I Solved Them

**Challenge 1: Looking at the wrong interface**
I initially searched for Extensions in the Cursor website on my browser instead of inside the installed Cursor application. After several minutes of confusion, I realized the mistake, switched to the desktop app, and found Extensions immediately.

Lesson: When instructions reference a tool, verify you're inside the tool itself, not its marketing website.

**Challenge 2: No progress indicator for Codex**
The Codex extension installation showed no percentage or progress bar, making it unclear whether it was installing or frozen. I resisted the urge to cancel and reinstall. After approximately 10 minutes, installation completed successfully.

Lesson: In async processes with no visible feedback, patience and observation beat intervention.

**Challenge 3: "Clone Repository" not in File menu**
The instructions suggested using File → Clone Repository in Cursor, but this option did not exist in my version. I opened the integrated terminal (Ctrl + `) and attempted git clone, which revealed that Git was not installed on my machine.

I installed Git for Windows, restarted Cursor, and successfully ran:
git clone https://github.com/andikasulyat/100hires-portfolio.git

Output confirmed successful clone:
"Cloning into '100hires-portfolio'... done."

Lesson: When a GUI option is missing, the terminal is the next step,
not the last resort.
